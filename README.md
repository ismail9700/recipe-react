A modernized version of the Forkify recipe application, rebuilt using React. This web app allows users to search for recipes, view detailed instructions, save favorites, and create custom recipes.

## Demo
[**Try Forkify-react Live**](https://forkify-r-i.netlify.app/)

## Screenshot
![App Screenshot](screenshots/forkify-screenshot.png)

## Features
- **Recipe Search**: Search for recipes using an external API.
- **Recipe Management**: Save favorite recipes and add custom recipes.
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Modern UI**: Sleek, React-powered interface with smooth interactions.
- **Dynamic Updates**: Real-time updates for search results and recipe details.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ismail9700/forkify-react-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd forkify-react-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## Usage
- Open the app in a browser to start exploring recipes.
- Use the search bar to find recipes by ingredients or keywords.
- Save favorite recipes or add your own custom recipes.
- Customize React components or CSS to modify functionality or styling.

## Technologies Used
- **React**: JavaScript library for building the user interface.
- **HTML5**: Structure for the web app.
- **CSS3**: Modern styling with responsive design.
- **JavaScript**: Handles API calls, state management, and dynamic updates.
- **External API**: Fetches recipe data (e.g., Forkify API).
